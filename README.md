# uni-assistant
### An Oshman Enterprises Venture

Simple LangChain/LangGraph implementation of a university FAQ chatbot. Classifies the user's query into one of five categories, one of which triggers a RAG pipeline to fetch relevant information from five .md files (stored in /data).
